Church year at a glance

Every day in twelve calendar months from the month of Advent Sunday — with or without liturgical observances — so you can scroll and compare.

Calendar defaults

Presets the transfer checkboxes below when you switch mode. Canonical matches the calendar table for optional moves. Pastoral matches typical parish defaults. You can still override any checkbox afterwards.

Easter date

Controls how Easter (and dependent feasts) are computed. Western matches the usual Church of England reckoning.

Transfers & calendar contents

Wording follows Common Worship. Hover a checkbox row to see the matching API property on CalendarOptions. All of these are applied on every refresh.

Local celebrations

Optional parish or cathedral extras — patronal festivals, dedications, harvest thanksgiving, and the like. Paste a JSON array; each entry needs at least a name, liturgical type, and ISO date (optional fields such as localType are allowed). Leave empty if you do not need any. Malformed JSON is shown as an error below. The library type is LocalEventInput.

It is not a substitute for the Ordinal or local customary where these differ.
